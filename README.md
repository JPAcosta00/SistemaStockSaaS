# 📦 Sistema de Gestión de Stock y Ventas

Aplicación web para administrar productos, controlar el stock y registrar ventas. 

## 🚀 Tecnologías


| Backend | Frontend |
|---------|----------|
| .NET (C#) | React |
| Entity Framework Core | Vite |
| MySQL |  |

## ✨ Funcionalidades

- Gestión de productos 
- Control de stock
- Registro de ventas
- Estadísticas e indicadores del negocio

## 🏛 Conceptos aplicados

- Programación Orientada a Objetos (POO)
- Clean Architecture
- Principios SOLID
- Repository Pattern
- Inyección de Dependencias
- Entity Framework Core
- API REST
