# 📦 Sistema de Gestión de Stock y Ventas

Aplicación web para administrar productos, controlar el stock y registrar ventas. 

## 🚀 Tecnologías

### Backend
- .NET (C#)
- Entity Framework Core
- MySQL

### Frontend
- React
- Vite

## ✨ Funcionalidades

- Gestión de productos (ABM)
- Control de stock
- Registro de ventas
- Estadísticas e indicadores del negocio

## 🏛 Conceptos aplicados

- Programación Orientada a Objetos (POO)
- Clean Architecture
- Principios SOLID
- Repository Pattern
- Inyección de Dependencias
- Entity Framework Core
- API REST
